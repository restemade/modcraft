# Ecopark 3D Concept Viewer

Интерактивное веб-приложение на Vite + React + TypeScript + Three.js для презентационного облёта концепции экопарка в режиме «квадрокоптера».

## Что учтено из исходных материалов

На текущем шаге в среде доступны только 2 изображения участка/окружения (скриншоты с контуром), поэтому:

- Контур участка восстановлен по видимым размерам и форме (`181м`, `355м`, `108м`, боковые сегменты).  
- В окружение заложен контраст между «урбанизированной курортной зоной» слева и «природной/болотистой» зоной справа.  
- Композиция построена как мастер-план демонстрационного экопарка: вход, общественное ядро, семейная/спортивная зоны, тихий сад, эко-набережная.

Если вы добавите `ТЗ`, `full_Region.jpg`, `area.png`, `ecopark presentation` и другие файлы в `public/uploaded/`, структуру можно быстро докрутить через `src/data/layout.ts`.

## Структура

- `src/app` — оболочка приложения, глобальный store, стили.
- `src/components` — Intro/HUD/Minimap/Loading overlay.
- `src/scene` — композиция сцены и слои (дорожки, вода, зелень, метки).
- `src/controls` — кастомный drone controller + auto-tour.
- `src/terrain` — генерация рельефа.
- `src/data` — конфигурация зон, контур, landmark-объекты, tour-точки.
- `src/config` — тема и параметры полёта.
- `src/utils` — вспомогательные функции (высота рельефа).
- `public/uploaded` — место для загруженных исходников.

## Управление

- `W/A/S/D` — движение
- `Mouse` — обзор
- `Space` — вверх
- `Shift` / `Ctrl` — вниз
- `Alt` — ускорение
- `Mouse wheel` — изменение базовой скорости
- `R` — сброс камеры
- `T` — автооблёт (autopilot)
- `M` — режим расширенной карты
- `H` — скрыть/показать HUD
- `Esc` — выход из pointer lock

## Запуск

```bash
npm install
npm run dev
```

Откройте URL из Vite (обычно `http://localhost:5173`).

## Production сборка

```bash
npm run build
npm run preview
```

## Куда класть ваши файлы

- `public/uploaded/full_Region.jpg`
- `public/uploaded/area.png`
- `public/uploaded/ecopark-presentation.pdf` (или набор изображений)
- дополнительные материалы — туда же

После этого можно уточнить:

- `src/data/layout.ts` (контур, зоны, ключевые объекты),
- `src/config/theme.ts` (палитра/бренд),
- `src/scene/ParkLayout.tsx` (арх-объекты и сценарии).
